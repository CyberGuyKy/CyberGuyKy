> **Portfolio Note**
>
> The projects in this portfolio are intentionally designed to **simulate the real workflows of an Identity & Access Management (IAM) professional and demonstrate knowledge**.
>
> Each project includes tools, protocols, and concepts such as **Active Directory, Microsoft Entra ID, Keycloak SIEM, Conditional Access, Access Reviews, Application Identity, SSO, MFA, Administrative Groups, etc**.
>
> The goal is to demonstrate **how a security engineer designs, configures, and manages identity systems across hybrid environments**, using scripting, identity platforms, and security best practices to enforce consistent access control.
